Simple ML program aiming to forecast the weather conditions in the metropolitan region of Porto Alegre by analysing past instances.

This program works by using a CSV file containing the weather conditions of the past 3 days plus the current day (in the ipynb, this CSV is labeled as "recent_data.csv"). This data must be gathered from the "INMET BDMEP" website, and it has to be from the meteorological station [A801], located in Porto Alegre, RS.

The program was developed with the assist of AI
